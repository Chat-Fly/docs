---
title: "Get Conversation History"
api: "GET https://backend.chatfly.co/api/chat/history"
description: "This endpoint get conversation history"
---

### Body
<ParamField body="bot_id" type="string">
  This is the ID of the bot you want to get the conversation history.
</ParamField>

<ParamField body="date_from" type="datetime">
  This is thetart date you want get conversation history.
</ParamField>

<ParamField body="date_to" type="datetime">
  This is the end date you want get conversation history.
</ParamField>

<ParamField body="order" type="string">
  This is the conversation sort option you want.
</ParamField>


### Response
<ResponseField name="JSON Response" type="string">
  Conversation information.
</ResponseField>

<RequestExample>

```bash Example Request
curl -X POST "https://backend.chatfly.co/api/chat/history" \
-H "Authorization: Bearer token" \
-H "Content-Type: application/json" \
-d '{
  "bot_id": "xxxx-xxxx-xxxx-xxxx",
  "date_from": "2023-01-01T00:00:00",
  "date_to": "2030-12-31T00:00:00",
  "order": "created_at.desc"
}'
```

</RequestExample>
