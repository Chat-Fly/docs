### API Endpoint

The API endpoint for obtaining a streaming response is:

<CodeGroup>
```text One
https://backend.chatfly.co/api/scraping/question-answer/{bot_id}
```

```text Many
https://backend.chatfly.co/api/scraping/question-answer/{bot_id}/many
```

</CodeGroup>

### Request Body

To import a website, you need to send a POST request to the API endpoint with a JSON request. Here's an example JSON request:

<CodeGroup>
```json One
{
  "user_id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "bot_id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "question": "string",
  "answer": "string"
}
```
```json Many
{
  "user_id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "bot_id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "question_answers": [
    {
      "question": "string",
      "answer": "string"
    }
  ]
}
```
</CodeGroup>

- bot_id (string): ID of your bot.
- user_id (string): ID of your account.
- question (string): Question.
- answer (string): Answer.

### Response Body

The streaming response is as follows:

<CodeGroup>
```json One
{
  "id": "string",
  "bot_id": "string",
  "question": "string",
  "answer": "string",
  "num_token": "int",
  "created_at": "datetime"
}
```

```json Many
[
  {
    "id": "string",
    "bot_id": "string",
    "question": "string",
    "answer": "string",
    "num_token": "int",
    "created_at": "datetime"
  },
  {
    ...
  }
]
```

</CodeGroup>

- id (string): ID of each question answer pair.
- bot_id (float): ID of your bot.
- question (string): Question
- answer (string): Answer
- num_token (int): Number of tokens consumed
