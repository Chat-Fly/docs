## API Endpoint

The API endpoint for obtaining a streaming response is:

<CodeGroup>
    ```text One
    https://backend.chatfly.co/api/scraping/question-answer/{bot_id}
    ```
    ```text Many
    https://backend.chatfly.co/api/scraping/question-answer/{bot_id}/many
    ```
</CodeGroup>

## Request Body

To import question-answer pairs, send a POST request to the API endpoint with a JSON request. Here's an example JSON request:

<CodeGroup>
    ```json Single Import
    {
        "bot_id": "your_bot_id",
        "question": "string",
        "answer": "string"
    }
    ```
    ```json Bulk Import
    {
      "bot_id": "your_bot_id",
      "question_answers": [
        {
          "question": "string",
          "answer": "string"
        },
        {
          "question": "string",
          "answer": "string"
        }
        // Add more Q-A pairs as needed
      ]
    }
    ```
</CodeGroup>

- bot_id (string): ID of your bot.
- question (string): Question.
- answer (string): Answer.

## Response Body

The streaming response is as follows:

<CodeGroup>
    ```json Single Import
    {
        "id": "string",
        "bot_id": "string",
        "question": "string",
        "answer": "string",
        "num_token": "int",
        "created_at": "datetime"
    }
    ```
    ```json Bulk Import
    [
        {
            "id": "string",
            "bot_id": "string",
            "question": "string",
            "answer": "string",
            "num_token": "int",
            "created_at": "datetime"
        }
        // Additional Q-A pairs
    ]
    ```

</CodeGroup>

- id (string): ID of each question-answer pair.
- bot_id (string): ID of your bot.
- question (string): Question
- answer (string): Answer
- num_token (int): Number of tokens consumed
