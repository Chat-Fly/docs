title: "Embed ChatFly on Your Site"
description: "In this guide, we will walk through the process **Embed ChatFly Bot on Your Site**"

---

To add the chatbot anywhere on your website, add this iframe to your html code, this iframe like

``` json
    <iframe
      src="https://app.chatfly.co/chat/xxxx-xxxx-xxxx-xxxx-xxxx"
      width="100%"
      height="300"
      style="border:1px solid black;"
    ></iframe>
```

To add a chat bubble to the bottom right of your website add this script tag to your html

``` json
    <script>
      window.chatbotConfig = {
        bot_id: "xxxx-xxxx-xxxx-xxxx-xxxx",
      };
    </script>
    <script src="https://app.chatfly.co/Chat.js"></script>
```

<Warning>

Replace `xxxx-xxxx-xxxx-xxxx-xxxx` with your bot-id.

</Warning>
