## Integration Steps

#### Step 2: Select Your Workspace

## Post-Installation Steps

- **Step 1:** Navigate to your Slack workspace, go to the 'Apps' section, and select 'Browse apps.'
![Browse Apps](https://app.chatfly.co/assets/2.1-e10dee78.png)

- **Step 2:** Search for the 'ChatFly' app and select it.
![Select ChatFly App](https://app.chatfly.co/assets/2.2-00de1b96.png)

- **Step 3:** In the 'ChatFly App About' section, find the command line for interacting with the chatbot.
![Command Line](https://app.chatfly.co/assets/2.3-8b9d01df.png)

- **Step 4:** Add the Chatbot to Slack using the bot_id obtained from the ChatFly App.
